### 코드 해석

```
import { ref } from 'vue'  
```
- frontend 할때 변수지정이다.
- 여기서 ref 는 반응형(변하는) 데이터를 만들때 사용한다(Vue)

```
defineProps({
  msg: String,
})
```
- 부모 컴포넌트에서 받은 데이터(props) 를 정의하는 부분


```
const count = ref(0)
```
- count 라는 숫자 변수를 만들고 초기값 0으로 설정

- reactive 변수

```
<button type="button" @mouseenter="count++">count is {{ count }}</button>
```
- 해석: 해당버튼영역에 마우스가 포함되면 count 값을 1증가 시켜 화면에 현재 count를 표시한다

- @touch, @mouseenter 등 이벤트 함수
- "count++" : count 값이 1 증가
